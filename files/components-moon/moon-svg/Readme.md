#  moon-svg

### 作者：闰月飞鸟；时间：2020/03/14
### 开发目的
-  封装svg图标组件。
### 设计思路
- svg图片统一放在一个目录下，通过props传递的svg图片名与size来展示图片和控制大小。

### Props 
参数 |说明|类型|可选值|默认值
---|---|---|---|---
name|/static/image/common/svg/目录下svg图片名|String|—|""
size |图片尺寸| Number|-|16

###  Events
名称 |说明| 参数
---|---|---|
click| 图片点击触发事件
