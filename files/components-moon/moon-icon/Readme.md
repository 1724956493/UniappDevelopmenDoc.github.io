# moon-icon

### 作者：闰月飞鸟；时间：2020/12/04
### 开发目的
- 获取远端图标库组件
 

### Props 
参数 |说明|类型|可选值|默认值
---|---|---|---|---
name|图标类名|String|-|
fontFamily|指定字体名|String |-|必填   内置了 moon-icon qichacha-icon
color|图标颜色|String|-|""
size|图标尺寸|String Number|-|inherit
bold|图标加粗|Boolean|-|false
hoverClass|鼠标钩子样式|String|-|''
href|外部https链接 css|String |-|''
